# More

`more` is a basic pager built-in to PowerShell.

### Commands
| Command  | Description                           |
|----------|---------------------------------------|
| `Enter`  | Display the next line of output       |
| `Space`  | Display the next screen of output     |
| `q`      | Quit and return to the command prompt |

### Example
```powershell
> ps | more
```

```powershell
> more <file>
```
